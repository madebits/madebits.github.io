#Reading a Text File from Command-line in Windows

2012-08-30

Different ways of reading a text `test.txt` file from command-line in Windows:

```
C:\>test.txt
C:\>start test.txt
C:\>notepad test.txt
C:\>more test.txt
C:\>type test.txt
C:\>copy test.txt con
```
