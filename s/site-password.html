PassGen: Web Site Specific Password

Generates a per web site specific password from a master password. You only need to remember the master password and then use something unique per each web site that is easy to find, such as, its domain. Using same input data generates same password.



Web Site Specific Password Alternatives
  • Base85
  • Base64
  • Base16
Choose one. Click to copy.

Password is generated locally (via JavaScript) using PBKDF2 ( iterations) over 'Master Password' using 'Web Site Domain' as salt. No data are sent to any server. Do do not use this page to generate very sensitive passwords. Same generated password is shown encoded in Base85, Base64, and Base16 with lengths 8, 16, and 32. You should normally prefer the biggest base (Base85) with longest length (32), unless the web site complains about the length or characters.
